Proyecto de Heurísticos

Optimización combinatoria aplicada a problemas clásicos

Descripción General

Este proyecto presenta la implementación de algoritmos heurísticos, metaheurísticos y exactos en Python para resolver problemas clásicos de optimización combinatoria.

Los problemas trabajados son el Problema de la Mochila 0/1 y el Problema del Agente Viajero. Además, el proyecto incluye pruebas unitarias, una libreta de Jupyter con ejemplos de uso y una estructura de paquete instalable.

Problema de la Mochila 0/1

El Problema de la Mochila 0/1 consiste en seleccionar un subconjunto de objetos, cada uno con un peso y un valor, con el objetivo de maximizar el valor total sin exceder una capacidad máxima. Cada objeto solo puede tomarse una vez o no tomarse.

Modelo Matemático

max Σ pᵢxᵢ sujeto a: Σ wᵢxᵢ ≤ M xᵢ ∈ {0,1}

Donde pᵢ representa el valor del objeto, wᵢ representa su peso, M es la capacidad máxima de la mochila y xᵢ indica si el objeto se selecciona o no.

Algoritmos Implementados

  • Greedy: selecciona objetos usando una estrategia basada en la relación valor/peso.
  • Backtracking: explora combinaciones posibles para encontrar una solución óptima.
  • Recocido Simulado: utiliza una metaheurística inspirada en procesos físicos de enfriamiento.

Complejidades

  • Greedy → O(n log n)
  • Backtracking → O(2ⁿ)
  • Recocido Simulado → depende del número de iteraciones configuradas

Ejemplo de Uso

from heuristicos_pkg.knapsack import ( mochila_greedy, mochila_backtracking, mochila_recocido_simulado ) valores = [60, 100, 120] pesos = [10, 20, 30] capacidad = 50 # Greedy solucion_greedy, valor_greedy = mochila_greedy(valores, pesos, capacidad) # Backtracking valor_optimo, peso_optimo, seleccion_optima = mochila_backtracking( pesos, valores, capacidad ) # Recocido simulado solucion_rs, valor_rs = mochila_recocido_simulado( valores, pesos, capacidad, iteraciones=1000 ) peso_rs = sum( pesos[i] for i in range(len(solucion_rs)) if solucion_rs[i] == 1 ) print("=== GREEDY ===") print("Solución:", solucion_greedy) print("Valor:", valor_greedy) print("\n=== BACKTRACKING ===") print("Solución:", seleccion_optima) print("Valor:", valor_optimo) print("Peso:", peso_optimo) print("\n=== RECOCIDO SIMULADO ===") print("Solución:", solucion_rs) print("Valor:", valor_rs) print("Peso:", peso_rs)

Resultados Esperados

Algoritmo Tipo Descripción
Greedy Heurístico Rápido, pero no siempre garantiza la solución óptima.
Backtracking Exacto Puede encontrar la solución óptima, pero es costoso para instancias grandes.
Recocido Simulado Metaheurístico Busca buenas soluciones mediante exploración aleatoria controlada.

Problema del Agente Viajero

El Problema del Agente Viajero, también conocido como TSP por sus siglas en inglés, consiste en encontrar una ruta de costo mínimo que visite cada ciudad exactamente una vez y regrese al punto de partida.

Este problema es importante porque aparece en aplicaciones como logística, transporte, planeación de rutas y optimización de recorridos. Para instancias grandes, encontrar la solución óptima puede ser muy costoso, por lo que se utilizan métodos heurísticos para obtener buenas soluciones en tiempos razonables.

Modelo Matemático

C(x) = Σ d(xₖ, xₖ₊₁) + d(xₙ, x₁)

Donde C(x) representa el costo total de la ruta y d(xₖ, xₖ₊₁) representa la distancia entre dos ciudades consecutivas.

Algoritmos Implementados

  • Generación aleatoria de matrices de costos.
  • Generación de rutas aleatorias.
  • Cálculo del costo total de una ruta.
  • Búsqueda local 2-OPT para mejorar rutas iniciales.

Complejidades

  • Generación de ruta aleatoria → O(n)
  • Generación de población de rutas → O(p · n), donde p es el tamaño de la población
  • Cálculo de costo de una ruta → O(n)
  • Búsqueda local 2-OPT → aproximadamente O(n³) en el peor caso

Ejemplo de Uso

from heuristicos_pkg.viajero import TSP # Matriz de costos M = [ [0, 10, 15, 20, 25], [10, 0, 35, 25, 30], [15, 35, 0, 30, 20], [20, 25, 30, 0, 15], [25, 30, 20, 15, 0] ] # Crear instancia del problema tsp = TSP(M) # Generar una población de rutas aleatorias poblacion = tsp.generar_poblacion(tamano_poblacion=3) # Mostrar rutas y costos for i, ruta in enumerate(poblacion): costo = tsp.calcular_costo(ruta) print(f"Ruta {i + 1}: {ruta}") print(f"Costo: {costo}") # Mejorar una ruta usando 2-OPT ruta_inicial = poblacion[0] ruta_mejorada, costo_mejorado = tsp.dos_opt(ruta_inicial) print("\n=== MEJORA CON 2-OPT ===") print("Ruta inicial:", ruta_inicial) print("Ruta mejorada:", ruta_mejorada) print("Costo mejorado:", costo_mejorado)

Resultados Esperados

Elemento Descripción
Ruta inicial Ruta generada aleatoriamente.
Costo inicial Costo total antes de aplicar mejora.
Ruta mejorada Ruta obtenida después de aplicar 2-OPT.
Costo mejorado Costo total después de la mejora local.
